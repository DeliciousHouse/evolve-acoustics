Loading Transition Test

This page demonstrates the smooth loading transition with the js-loading class. When the page loads, the preloader is displayed and fades out once content is ready. Then the js-loading class is removed, making the content visible with a smooth transition.

Here's how it works:

  1. HTML starts with the js-loading class on the html element
  2. CSS hides the body when js-loading class is present
  3. Preloader is shown while the page loads
  4. Once loaded, the preloader fades out over 0.8 seconds
  5. After fade-out completes, js-loading class is removed
  6. Body becomes visible with a smooth transition
